HTML Cheatsheet

HTML tags

Tag Name Description
<ol> Ordered list The <ol> ordered list element creates a list of items in sequential order. Each list item appears numbered by default.
<ul> Unordered list The <ul> unordered list element is used to create a list of items in no particular order.
<li> List The list item element create list items inside: Ordered lists <ol> Unordered lists <ul>
<video> Video The <video> element embeds a media player for video playback.
<em> Emphasis Element The <em> emphasis element emphasizes text and browsers will usually italicize the emphasized text by default.
<div> Div Element The <div> element is used as a container that divides an HTML document into sections and is short for “division”.
<br> Line Break Element The <br> line break element will create a line break in text and is especially useful where a division of text is required, like in a postal address.
<img> Image Element HTML image <img> elements embed images in documents.
<h1>-<h6> <h1>-<h6> Heading Elements HTML can use six different levels of heading elements.
<p> Paragraph Element The <p> paragraph element contains and displays a block of text.
<body> Body Element The <body> element represents the content of an HTML document.
<span> Span Element The <span> element is an inline container for text and can be used to group text for styling purposes.
<strong> Strong Element The <strong> element highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default.
